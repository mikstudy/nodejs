##### 1. Uglify-Js
 `npm install uglify-js`
- 코드 압축 및 정리 모듈
- 사용법
```
$ uglifyjs "[Input]" -o "[Output]" -c "[옵션]" -m "[옵션]"
```
- [옵션] 설명 - http://www.sangkle.com/archives/580
